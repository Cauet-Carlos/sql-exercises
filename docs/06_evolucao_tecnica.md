# Evolução Técnica

Este repositório foi desenvolvido com foco em evolução progressiva em SQL analítico.

Os exercícios foram organizados por complexidade e conceitos técnicos.

---

# Etapa 1 — Fundamentos SQL

Conceitos praticados:

- SELECT
- WHERE
- ORDER BY
- DISTINCT
- LIMIT
- LIKE
- BETWEEN
- IS NULL

Aprendizados:
- filtragem de dados;
- ordenação;
- consultas básicas;
- manipulação inicial de tabelas.

---

# Etapa 2 — JOINS

Conceitos praticados:

- INNER JOIN
- LEFT JOIN

Aprendizados:
- relacionamento entre tabelas;
- cruzamento de informações;
- construção de métricas analíticas;
- identificação de registros inexistentes.

---

# Etapa 3 — Análise Temporal

Conceitos praticados:

- agrupamentos temporais;
- crescimento percentual;
- acumulados;
- médias móveis.

Aprendizados:
- análise histórica;
- comportamento temporal;
- evolução de métricas;
- tendências de faturamento.

---

# Etapa 4 — Subqueries

Conceitos praticados:

- subqueries no WHERE;
- EXISTS;
- NOT EXISTS;
- subqueries no FROM.

Aprendizados:
- consultas aninhadas;
- filtros avançados;
- comparações analíticas;
- reutilização de resultados.

---

# Etapa 5 — CTEs

Conceitos praticados:

- WITH;
- múltiplas CTEs;
- pipelines analíticos.

Aprendizados:
- modularização;
- organização lógica;
- reaproveitamento de consultas;
- redução de complexidade.

---

# Etapa 6 — Window Functions

Conceitos praticados:

- RANK();
- DENSE_RANK();
- ROW_NUMBER();
- LAG();
- LEAD();
- NTILE();
- FIRST_VALUE();
- funções OVER().

Aprendizados:
- análises avançadas;
- segmentações;
- rankings;
- comparações temporais;
- análises acumuladas.
